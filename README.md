## Installation

YMTCloudClassroom is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'YMTBigClassRoom', :git => 'https://github.com/SendomLog/YMTBigClassRoom.git'
```
